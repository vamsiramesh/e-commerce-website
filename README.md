# E-Commerce Catalog

A simple React + Vite e-commerce catalog application developed as Project 5.

## Features

- Home page
- Product listing
- About page
- Reusable Navbar component
- Reusable Product Card component
- Responsive UI
- Fast development using Vite

## Technologies Used

- React
- Vite
- JavaScript (JSX)
- HTML
- CSS

## Project Structure

```
src/
├── components/
│   ├── Navbar.jsx
│   └── ProductCard.jsx
├── pages/
│   ├── Home.jsx
│   ├── Product.jsx
│   └── About.jsx
├── App.jsx
├── main.jsx
├── App.css
└── index.css
```

## Installation

Clone the repository:

```bash
git clone https://github.com/your-username/ecommerce-catalog.git
```

Go to the project folder:

```bash
cd ecommerce-catalog
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Open your browser and visit:

```
http://localhost:5173
```
